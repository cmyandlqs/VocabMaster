> A beautiful Mac-style vocabulary learning PWA for postgraduate exam preparation

## ✨ Features

- **macOS Big Sur Style UI** - Glassmorphism, squircle corners, blend of skeuomorphic and flat design
- **PWA Support** - Add to iOS/Android home screen, fullscreen app experience
- **Offline Ready** - Service Worker caching, works without internet
- **Smart Study Plan** - Daily goals, step-by-step learning
- **Random Quiz** - Test your memory after each chapter
- **Text-to-Speech** - Built-in English pronunciation

## 📱 How to Install

### iOS

1. Open the app URL in Safari
2. Tap the share button at the bottom
3. Scroll down and select "Add to Home Screen"
4. Tap "Add"

### Android

1. Open the app URL in Chrome
2. Tap the install prompt in the address bar
3. Or tap menu → "Add to Home Screen"

## 🛠 Tech Stack

- HTML5 + CSS3 + JavaScript (ES6+)
- PWA (Service Worker + Manifest)
- LocalStorage for data persistence
- Web Speech API for pronunciation

## 📂 Project Structure

```
├── index.html          # Main entry
├── manifest.json       # PWA manifest
├── service-worker.js   # Offline caching
├── css/                # Stylesheets
├── js/                 # JavaScript files
└── assets/             # Static assets
```
