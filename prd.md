## 2. 🎨 Mac OS 风格 UI/UX 设计规范

> *"大量使用毛玻璃（亚克力）半透明材质，背景模糊效果。卡片和按钮使用苹果的平滑大圆角。整体要求极简、克制、优雅，排版呼吸感强，无多余线条，靠弥散阴影区分层级。"*

### 2.1 色彩系统 (Color Palette)

```css
/* ==================== 浅色模式 ==================== */
```

## 5. 数据结构

### 5.1 单词 (Word)

```typescript
interface Word {
  id: string;                    // 唯一标识
  word: string;                 // 单词本体
  phonetic: string;             // 音标
  phoneticAudio?: string;       // 发音音频 URL
  definitions: Definition[];     // 释义列表
  examples: Example[];           // 例句
  roots?: string;               // 词根词缀
  memory?: string;              // 助记技巧
  tags: string[];               // 标签 ["红宝书", "核心词"]
  masterLevel: MasterLevel;     // 掌握程度
}

interface Definition {
  part: string;                 // 词性 (n./v./adj.)
  meaning: string;             // 释义
}

interface Example {
  sentence: string;             // 英文例句
  translation: string;          // 中文翻译
  source?: string;              // 来源 (历年真题等)
}

type MasterLevel = 0 | 1 | 2 | 3;
// 0: 未学习 1: 认识 2: 模糊 3: 掌握
```

### 5.2 学习记录 (StudyRecord)

```typescript
interface StudyRecord {
  wordId: string;
  lastReview: number;           // 上次复习时间戳
  nextReview: number;           // 下次复习时间戳
  correctCount: number;         // 正确次数
  wrongCount: number;           // 错误次数
  status: 'new' | 'learning' | 'reviewing' | 'mastered';
  interval: number;             // 复习间隔(天)
}
```

### 5.3 学习计划 (StudyPlan)

```typescript
interface StudyPlan {
  id: string;
  name: string;                // 计划名称
  libraryId: string;           // 关联词库 ID
  dailyGoal: number;           // 每日目标 (词数)
  startDate: number;           // 开始日期
  totalDays: number;           // 总天数
  completedDays: number;       // 已完成天数
  chapters: Chapter分配[];      // 章节分配
  status: 'active' | 'paused' | 'completed';
}

interface Chapter分配 {
  chapterId: string;
  targetDate: string;          // 目标日期 YYYY-MM-DD
  wordIds: string[];           // 该天要学的单词 IDs
  completed: boolean;
}
```

```typescript
  wordIds: string[];           // 单词 ID 列表
  source: 'builtin' | 'custom'; // 内置/自定义
  createdAt: number;           // 创建时间
}
```

### 6.3 Service Worker 缓存策略

```javascript
// 缓存策略
const CACHE_NAME = 'vocab-master-v1';
const ASSETS = [
  '/',
  '/index.html',
  '/css/style.css',
  '/js/app.js',
  '/data/vocabulary.json',
  '/assets/icon-192.png',
  '/assets/icon-512.png'
];

// 优先缓存，缓存不到再请求网络
self.addEventListener('fetch', event => {
  event.respondWith(
    caches.match(event.request).then(response => {
      return response || fetch(event.request);
    })
  );
});
```

```
Unit 2:  B - bachelor ~ bustle

Unit 26: Z - zeal ~ zoom
```
